Summary: Latitude vs. X

This objective of this project is to create a visualization dashboard website of the Matplotlib plots that are created as part of the Python API assignment.
The project starts with collecting the weather data of randomly selected over 500 cities around the world from Openweather API. The data is then analysed on different latitudes against selected westher parameters like temperature, humidity, cloudiness, and wind speed. This site provides the sourse of data and visualizations that are created as part of the analysis along with teh explaination and description of any trends and correlation witnessed.
